Sample AMC 8 Style Problem

Here's an example of the type of problem our Stage 2 students master:

Number Theory Challenge

The sum of the digits of a two-digit number is 12. When the digits are reversed, the new number is 18 more than the original number. What is the original number?

Answer choices: (A) 39 (B) 48 (C) 57 (D) 75 (E) 84

Solution Approach:

Let the original number be 10a + b, where a and b are the tens and units digits.

Given: a + b = 12 and (10b + a) - (10a + b) = 18

Simplifying: 9b - 9a = 18, so b - a = 2

Solving the system: a = 5, b = 7, so the original number is 57.
