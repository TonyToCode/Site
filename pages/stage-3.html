Sample Problems

AMC 10 Focus Track

Problem:

How many three-digit numbers with all different digits have digits summing to 18 and increase by 198 when reversed?

A) 1
B) 2
C) 3
D) 4
E) 5

Teaching Approach:

Start by identifying the key conditions: digits sum to 18, reversed number is 198 greater, and digits are different.

Set digits as a, b, c and write the equation for the difference when reversed: 99(c - a) = 198, so c - a = 2.

Use the sum condition: a + b + c = 18, substitute c = a + 2 to get b = 16 - 2a.

Check values of a from 1 to 9, only keep those where b and c are digits 0–9.

Count all valid triples (a, b, c) that satisfy these and have distinct digits.

The answer is the total number of such numbers.

AMC 12 Focus Track

Problem:

A sequence of numbers is defined as follows: a₁=2, and for every n≥1,

aₙ₊₁ = 3aₙ + 1

What is the remainder when a₁₀ is divided by 7?

A) 0
B) 1
C) 2
D) 3
E) 4

Teaching Approach:

Start by understanding the recursive formula aₙ₊₁ = 3aₙ + 1.

Explain the problem asks for the remainder of a₁₀ mod 7, so introduce modular arithmetic.

Calculate the first few terms modulo 7 to identify a pattern or cycle.

Show how to use patterns or the concept of modular cycles to avoid large computations.

Guide students to find a₁≡2 mod 7, then a₂, etc., until they detect a repeating pattern.

Use the pattern to find the remainder of a₁₀ without calculating the full number.

Encourage checking work carefully to avoid mistakes in modular arithmetic.
